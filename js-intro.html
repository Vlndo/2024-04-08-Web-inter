Javascript Intro

JS est un langage de programmation FRONT, c'est à dire qu'il sert à ajouter de l'interaction avec le visiteur sans passer par une compilation serveur.
C'est le navigateur qui possède un moteur de rendu qui va compiler et exécuter le langage.

Le JS est capable d'agir sur les élément HTML de la page, de faire appel à des API pour récupérer des données sur des ressources tiers (site, serveur) et de les disposer à l'affichage sans avoir a recharger la page côté utlisateur.
